# Спрайты зданий

Кинь сюда свои картинки — они подменят эмодзи. Имя файла = тип здания.
Сервер сканирует папку при старте и при каждом входе игрока, так что можно
просто добавить файл и обновить вкладку (рестарт не нужен).

Форматы: `.png`, `.webp`, `.jpg`, `.gif`, `.svg`. Рекомендуется квадрат
(например 64×64 или 128×128), прозрачный фон для PNG.

Имена файлов (без картинки рисуется эмодзи по умолчанию):

| Файл | Здание |
|------|--------|
| `house.png`   | Дом 🏠 |
| `road.png`    | Дорога 🛣 |
| `well.png`    | Колодец 💧 |
| `farm.png`    | Ферма 🌾 |
| `clinic.png`  | Клиника 🏥 |
| `police.png`  | Полиция 🚓 |
| `park.png`    | Парк 🌳 |
| `cafe.png`    | Кафе ☕ |
| `shop.png`    | Магазин 🏪 |
| `gym.png`     | Спортзал 🏋 |
| `school.png`  | Школа 🏫 |
| `theater.png` | Театр 🎭 |

Не обязательно закрывать все — что положишь, то и подменится, остальное
останется эмодзи.

## Тайлы ландшафта

Свои тайлы земли клади в подпапку `public/sprites/terrain/`. Имя файла = тип:

| Файл | Тайл |
|------|------|
| `grass.png`  | Трава (застраивается) |
| `forest.png` | Лес (застраивается, +счастье соседям) |
| `water.png`  | Вода (непроходимо, +счастье соседям) |
| `stone.png`  | Скалы (непроходимо) |

Форматы те же (png/webp/jpg/gif/svg), лучше квадрат под размер клетки. Нет
файла — тайл рисуется сплошным цветом. Сервер сканирует папку при входе игрока,
рестарт не нужен.
